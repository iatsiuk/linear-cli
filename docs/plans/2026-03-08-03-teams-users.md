# Teams & Users: Team and User Management

## Overview
- Implement read-only commands for teams and users, plus `linear me` (viewer)
- Commands: `linear team list`, `linear team show`, `linear user list`, `linear user show`, `linear me`
- Depends on Foundation plan

## Context
- Team key fields: id, name, displayName, description, icon, color, key, cyclesEnabled, issueEstimationType
- User key fields: id, email, displayName, avatarUrl, active, admin, guest
- `viewer` query returns current authenticated user (User type)
- Teams and users are read-only via CLI (no create/update needed for initial version)

## Technical Details

### Table columns
**team list**: Name | Key | Description | Cycles Enabled
**user list**: Name | Email | Role | Active

### Viewer query
```graphql
query { viewer { id email displayName active admin guest createdAt teams { nodes { id name key } } } }
```

### Viewer shortcuts
```graphql
query { viewer { assignedIssues { nodes { id identifier title state { name } } } } }
query { viewer { createdIssues { nodes { id identifier title state { name } } } } }
```
