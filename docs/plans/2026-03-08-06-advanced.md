## Technical Details

### Batch update mutation
```graphql
mutation IssueBatchUpdate($ids: [UUID!]!, $input: IssueUpdateInput!) {
  issueBatchUpdate(ids: $ids, input: $input) {
    issues { id identifier title state { name } }
  }
}
```

### Search query
```graphql
query SearchIssues($query: String!, $first: Int) {
  searchIssues(term: $query, first: $first) {
    nodes { id identifier title state { name } team { name } }
  }
}
```

### Date format for filters
Linear API natively supports ISO 8601 durations in date comparators:
- `P2W` -> 2 weeks from now
- `-P2W` -> 2 weeks ago
- `-P30D` -> 30 days ago
- `-P2W1D` -> 2 weeks and 1 day ago
- `2026-03-01` -> exact date (ISO 8601)
- `2021` -> midnight Jan 1, 2021

CLI convenience aliases (converted before sending to API):
- `today` -> current date ISO 8601
- `yesterday` -> yesterday's date ISO 8601
- `7d` -> `-P7D`
- `2w` -> `-P2W`
- `1m` -> `-P1M`
